# Gantz Homebrew Tap

Homebrew formulae for [Gantz CLI](https://github.com/gantz-ai/gantz-cli).

## Installation

```bash
brew install gantz-ai/tap/gantz
```

Or tap first:

```bash
brew tap gantz-ai/tap
brew install gantz
```

## Usage

```bash
gantz serve    # Start MCP server with gantz.yaml
gantz version  # Check version
```
